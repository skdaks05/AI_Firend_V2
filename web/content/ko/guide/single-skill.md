# 유즈케이스: 단일 스킬

## 시나리오

컴포넌트 1개 또는 API 1개처럼 범위가 좁은 작업.

## 프롬프트 예시

```text
Tailwind CSS와 검증 로직이 포함된 로그인 폼을 만들어줘.
```

## 예상 흐름

1. 관련 스킬 1개 자동 선택
2. 에이전트가 구현 결과 반환
3. 후속 개선 반복

## 팁

- 스택/제약을 구체적으로 명시
- 핵심 동작은 테스트 요구
- 첫 요청은 범위를 좁게 유지
