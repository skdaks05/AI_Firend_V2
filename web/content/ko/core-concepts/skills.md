# Skills

## Progressive Disclosure

요청 의도에 따라 스킬이 자동 선택됩니다. 대부분의 경우 수동 선택이 필요 없습니다.

## 2계층 설계

- `SKILL.md`: 정체성, 라우팅, 핵심 규칙
- `resources/`: 실행 프로토콜, 예시, 체크리스트, 플레이북, 스니펫

## 공통 리소스 레이어

`_shared/`에 재사용 프로토콜을 집중합니다.

- 추론 템플릿
- 모호성 해소 프로토콜
- 컨텍스트 예산 전략
- 라우팅 레퍼런스
- 검증 체크리스트

## 설계 효과

초기 컨텍스트를 가볍게 유지하면서 필요 시 깊은 실행을 지원합니다.
